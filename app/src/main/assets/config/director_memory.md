# Mémoire du metteur en scène

Ce fichier contient les préférences de Léandre accumulées au fil des sessions.
Il est lu à chaque génération et injecté dans le prompt de l'Orchestrateur.
**Modifier librement depuis GitHub pour affiner le comportement.**

## Préférences générales

- Privilégier les mouvements amples plutôt que timides
- Le regard vers le bas fonctionne bien pour les moments mélancoliques ou intérieurs
- Varier systématiquement la direction du regard — éviter de rester face caméra trop longtemps
- Les entrées et sorties de cadre donnent du dynamisme aux transitions
